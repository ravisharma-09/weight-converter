Weight Converter

This is a simple web project that converts pounds to kilograms. The user enters weight in pounds and the result in kilograms appears instantly.

I built this project using HTML CSS and JavaScript to practice user input handling and updating values on the screen in real time.

This small project helped me understand basic DOM manipulation and simple calculations in JavaScript.
